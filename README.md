# AI-Powered Personalized News Digest

## Overview
A MERN stack application that generates a personalized daily news digest using AI summarization based on user interests.

## Tech Stack
- React.js
- Node.js, Express.js
- MongoDB Atlas
- NewsAPI
- Groq (LLaMA-3.1-8B-Instant)

## Features
- User interest onboarding
- AI-powered news summarization
- Cleaned and preprocessed article content
- MongoDB-based storage
- In-app daily digest display

## Screenshots
- Frontend digest view
- Backend API response
- MongoDB stored summaries

## How to Run
1. cd server && npm start
2. cd client && npm start
